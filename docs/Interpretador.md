# Interpretador do Python

> executa o modulo
```
python mod.py 
```

> executa o modulo (unbuffered) do arquivo
```
python -u file.py
```

> executa o modulo biblioteca instalada como mod em script
```
python -m mod 
```

> comando para scripts
```
python -c 'print("ok");print(1+1)' 
```

> interativo com mod
```
python -i mod.py 
```
